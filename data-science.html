Customer Segmentation Using Hybrid Clustering (KM_AE)

View on GitHub  |  Live Streamlit Dashboard

A capstone project for retail customer segmentation using RFM features and a hybrid AutoEncoder + KMeans approach, packaged into an interactive Streamlit dashboard.

  • Model selection: Evaluated 6 model variants and selected KM_AE for the best clustering quality (Silhouette Score: 0.62).
  • Validation: Cluster separation validated via ANOVA + Tukey HSD and tested for robustness under noise/feature ablation.
  • Business impact: Personas (VIP, re-engagement, deprioritize) with estimated +15% retention improvement and +18% campaign ROI uplift.

SMS Spam Classifier (Streamlit)

View on GitHub

A Streamlit web app that classifies SMS/email messages as Spam or Not Spam using a TF-IDF vectorizer and a trained voting classifier.

  • Text preprocessing: lowercasing, stopword removal, and stemming (NLTK).
  • Model inference using scikit-learn with a saved TF-IDF vectorizer and voting classifier.
  • Deployable app workflow with clear inputs/outputs for end users.

Diabetes Prediction & Analysis (Machine Learning)

View on GitHub

A predictive modeling project comparing Logistic Regression, KNN, and Random Forest for diabetes prediction, supported by descriptive statistics and visualizations.

  • Best model: Random Forest achieved the highest performance (Accuracy: 97.22%, F1: 0.985, AUC: 0.985).
  • Key predictors: BMI, HbA1c levels, and blood glucose were the most significant features.
  • Included descriptive analysis such as geographic trends and model evaluation metrics (Accuracy, F1, AUC).
